# 📁 파일 형식 변환기

> 길종합건축사사무소 내부용 파일 변환 도구

이미지, 한글(HWPX), 엑셀 파일을 원하는 형식으로 간편하게 변환합니다.

---

## ⬇️ 다운로드

**[FileConverter Setup 1.0.0.exe 다운로드](https://github.com/Lejuwon/file-converter/releases/tag/v1.0.0)**

> Windows 전용 프로그램입니다.

---

## ✨ 주요 기능

| 파일 종류 | 변환 가능 형식 |
|-----------|---------------|
| 이미지 (PNG, JPG 등) | PNG, JPG |
| 한글 (HWPX) | PDF, DOCX |
| 엑셀 (XLSX, XLS 등) | PDF, DOCX |
| PDF | 그대로 저장 |

- 📂 저장 폴더 지정 가능 (지정 시 자동 저장)
- 🔢 동일 파일명 중복 시 자동으로 `(1)`, `(2)` 붙여서 저장
- 🖼️ HWPX 내 이미지 포함 변환 지원
- ⚠️ HWP 바이너리는 미지원 → 한컴오피스에서 HWPX로 저장 후 사용

---

## 🚀 사용 방법

1. 설치 파일 실행 후 설치
2. 파일을 드래그하거나 클릭하여 업로드
3. 출력 형식 선택 (파일 타입에 맞는 형식만 표시됨)
4. **변환하기** 버튼 클릭
5. 저장 폴더에 자동 저장 (미설정 시 기본 다운로드 폴더)
